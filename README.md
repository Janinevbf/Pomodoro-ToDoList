# Pomodoro e To Do List

## Funcionalidades
- Pomodoro 25 minutos 

- Pausa curta de 10 minutos

- Pausa longa de 15 minutos

- Adicionar tarefas
- Marcar tarefas como concluídas

- Remover tarefas

- Validação do input (não permite tarefas vazias)

##Tecnologias utilizadas

- React
- TypeScript
- TailwindCSS
- Vite
